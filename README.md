# YaLyceumGram - Социальная сеть для студентов

## Автор проекта
Назаров Никита 

---

## Описание проекта

YaLyceumGram - это веб-приложение (микроблог с элементами социальной сети), разработанное на фреймворке Flask. Проект представляет собой платформу, где пользователи могут публиковать посты, оставлять комментарии, ставить лайки, обмениваться личными сообщениями и видеть друг друга на интерактивной карте. (Название очень долго придумывал)

---

## Актуальность

Современные студенты проводят много времени в интернете и нуждаются в удобной платформе для общения и самовыражения. Существующие социальные сети перегружены рекламой и отвлекающим контентом.

YaLyceumGram предлагает:
- Простой и чистый интерфейс без рекламы
- Возможность публиковать мысли и идеи
- Общение с однокурсниками в личных чатах
- Визуализацию расположения друзей на карте
- Полный контроль над своим контентом

---

## Функционал программы

1. Регистрация и авторизация - создание аккаунта с хешированием паролей
2. Лента постов - просмотр всех публикаций пользователей
3. Создание постов - публикация текста с прикреплением изображений
4. Комментарии - оставление комментариев под постами
5. Лайки - оценка понравившихся публикаций
6. Личные сообщения - приватный чат между двумя пользователями
7. Карта пользователей - отображение участников на Яндекс.Картах
8. Профиль пользователя - просмотр и редактирование информации, аватарка
9. Адаптивный дизайн - корректное отображение на любых устройствах

## Технические особенности

- Бэкенд: 
- База данных: SQLite + SQLAlchemy ORM
- Шаблоны: Jinja2 + Bootstrap 5
- Формы: Flask-WTF
- Авторизация: Flask-Login
- Карты: Яндекс.Карты Static API
- Геокодирование: Яндекс.Геокодер API
- Хеширование паролей: Werkzeug

## Классы и модули программы

### Модели данных (папка data)

1. User - модель пользователя (имя, почта, пароль, город, аватарка)
2. Post - модель поста (заголовок, текст, картинка, лайки, комментарии)
3. Message - модель сообщения (текст, отправитель, получатель)
4. Comment - модель комментария (текст, автор, пост)
5. Like - модель лайка (пользователь, пост)

### Формы (папка forms)

1. RegisterForm - форма регистрации
2. LoginForm - форма входа
3. EditProfileForm - форма редактирования профиля с загрузкой аватарки
4. PostForm - форма создания поста с загрузкой картинки
5. CommentForm - форма написания комментария

### Шаблоны (папка templates)

1. base.html - базовый шаблон с навигацией и футером
2. index.html - лента постов
3. register.html - страница регистрации
4. login.html - страница входа
5. add_post.html - создание нового поста
6. post_detail.html - просмотр поста с комментариями
7. profile.html - профиль пользователя
8. edit_profile.html - редактирование профиля
9. map.html - карта пользователей
10. messages.html - список диалогов
11. chat.html - личный чат

## База данных

Приложение использует SQLite базу данных с пятью таблицами:

- users - хранение информации о пользователях
- posts - хранение постов
- messages - хранение личных сообщений
- comments - хранение комментариев
- likes - хранение лайков

Таблицы связаны между собой через внешние ключи. Например, каждый пост привязан к пользователю через user_id, а каждый комментарий - к посту через post_id.

## Использованные технологии и библиотеки

1. Flask - веб-фреймворк
2. Flask-WTF - работа с формами
3. Flask-Login - авторизация пользователей
4. SQLAlchemy - ORM для работы с базой данных
5. SQLAlchemy-serializer - сериализация моделей в JSON
6. Jinja2 - шаблонизатор
7. Bootstrap 5 - CSS-фреймворк для стилизации
8. Requests - HTTP-запросы к API Яндекс.Карт
9. Werkzeug - хеширование паролей
10. SQLite - встроенная база данных

## Инструкция по установке и запуску

1. Установить Python 3.8 или выше
2. Установить зависимости: pip install -r requirements.txt
3. Запустить приложение: python main.py
4. Открыть в браузере: http://127.0.0.1:8080
